Compute the real-valued FFT of a time series.

  • fs (float) – Sampling frequency (Hz).

  • remove_mean (bool, default True) – If True, subtract the mean before FFT.

  • periodogram (bool, default False) – If True, return 1/freq and FFT magnitudes (excluding zero-frequency term).
